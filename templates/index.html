Welcome to the 8th Army Engagement Heat Map Application

This application creates an interactive map of South Korea by reading engagement data from an Excel file and plotting the results as a heatmap. In addition, it displays static markers to show open source locations for ROK and US military bases. Red Stars represent US installations and Blue Circles represent ROK installations.

• The file must be in .xlsx format.
• It should have two columns without headers:
  • Column A: Location Name (e.g., a city or military base in South Korea)
  • Column B: Engagement Count (a numeric value)

Simply upload your Excel file and click “Generate Map” to view your map.
